# Hi, I'm David

I’m a data analyst with a background in accounting and auditing. Working extensively with financial and operational data gave me a strong foundation in data structures, internal controls, and governance.

As datasets became larger and more complex, I began applying computer-assisted audit techniques (CAATs) to increase analytical coverage and improve efficiency. This evolved into a dedicated analytics focus, where I build SQL and Python solutions, automate workflows, and strengthen data validation processes.

Here you’ll find some of my projects that demonstrate my skills, the tools I use and how I communicate my results.

If you’re interested in my professional background, you can find more context on [LinkedIn](https://www.linkedin.com/in/davidgriesel).
